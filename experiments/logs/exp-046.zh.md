## EXP-046: GateAttention — Sigmoid Gate on Attention Output

[English](exp-046.md) | [中文](exp-046.zh.md)

[English](exp-046.md) | [Chinese](exp-046.zh.md)

**Date**: 2026-04-29
**Status**: completed

### Background
在 TransformerLayer 的 attention output 上加 sigmoid gate：
`attn_out = attn_out * sigmoid(W_g * x_norm)`，W_g ∈ R^{D×D}，新增参数约 +1%（6 × 256² ≈ 393K）。
假设：per-position gating 能抑制噪声 token 的注意力影响，提升序列建模质量。

### Design
- **Variable**: 有无 gate_attn
- **Fixed**: S-tier + 0.6B SID，14d 数据，segment_emb + time_gap + action_level
- **Baseline**: exp043-s-0.6b（全量 eval R@500=61.2%，PPL=26.52）

| Config | 说明 |
|--------|------|
| exp043-s-0.6b | Baseline |
| exp046-gate-attn | +gate_attn |

### Results

| Config | R@10 | R@500 | PPL |
|--------|------|-------|-----|
| exp043-s-0.6b (baseline) | 11.4% | 61.2% | 26.52 |
| **exp046-gate-attn** | **10.2%** | **61.6%** | **26.07** |

### Analysis

1. **R@500 +0.4pp**（61.2% → 61.6%），PPL 略降（26.52 → 26.07），正向但边际效益小。
2. **R@10 下降**（11.4% → 10.2%）：top-10 精度轻微下滑，说明 gate 可能影响了高置信度预测的排序，但长尾召回（R@500）有改善。
3. **结论**：GateAttention 有轻微正向作用，计算成本低（+1% 参数），可作为默认组件保留，但不是决定性因素。
